# TermFun
A Termux Backdoor 

TermFun was developed to show off what you can do with the Termux-api

Install :

1. Open Termux
2. pkg install git 
3. git clone https://github.com/f00kthisc0de/TermFun
4. cd TermFun
5. chmod +x *
6. ./installer.py
7. python3 TermfunListener.py
8. connect to your phone with nc (your phones ip) 31338
9. Have Fun!
